## Function-Transformer in Machine Learning

-> This project demonstrates how to use "Scikit-learn’s FunctionTransformer" to apply "custom preprocessing functions" to datasets.
-> It’s useful when you want to integrate your own transformation logic directly into a pipeline or ML workflow.


# Topics Covered:- 
-> Introduction to `FunctionTransformer`
-> Creating and applying custom functions
-> Transforming features using mathematical operations
-> Integrating FunctionTransformer with pipelines


# Libraries Used:-
-> pandas  
-> numpy  
-> scikit-learn  


# Key Takeaway:-
`FunctionTransformer` allows developers to:
   -> Apply "custom feature transformations" 
   -> Maintain a "consistent workflow within scikit-learn  
   -> Improve code "modularity and reusability"


=> Note:- The notebook includes examples of using `FunctionTransformer` for mathematical transformations and preprocessing integration.
